📌 Online Recruitment Fraud Detection Using Deep Learning
📖 Overview

This project focuses on detecting fraudulent job postings using advanced deep learning and Natural Language Processing (NLP) techniques. With the increasing number of online job scams, this system aims to classify job advertisements as real or fake, improving recruitment platform safety.

🎯 Objective
To identify and prevent online recruitment fraud
To build an intelligent classification model using deep learning
To improve detection accuracy on imbalanced datasets
🛠️ Technologies Used
Python
Natural Language Processing (NLP)
BERT (Bidirectional Encoder Representations from Transformers)
RoBERTa
SMOTE (Synthetic Minority Over-sampling Technique)
Scikit-learn, Pandas, NumPy
⚙️ Methodology
Data Collection: Dataset containing real and fraudulent job postings
Data Preprocessing: Cleaning text, removing noise, tokenization
Handling Imbalance: Applied SMOTE to balance dataset
Model Building: Implemented BERT and RoBERTa models
Training & Testing: Split dataset and trained models
Evaluation: Measured accuracy, precision, recall, and F1-score
📊 Results
Achieved high accuracy in detecting fraudulent job postings
Improved recall using SMOTE for better fraud detection
Transformer models outperformed traditional machine learning approaches
🚀 Key Features
Detects fake job postings automatically
Uses advanced transformer-based NLP models
Handles imbalanced datasets effectively
Scalable for real-world applications
🔍 Future Scope
Integration with real-time job portals
Use of ensemble deep learning models
Deployment as a web application or API
Continuous learning with updated datasets
📌 Conclusion

The project successfully demonstrates how deep learning techniques can be used to detect online recruitment fraud. It provides a reliable and scalable solution to improve trust and safety in online job platforms.
